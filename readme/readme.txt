plugin for CudaText.
supports building Code Tree for Markdown lexer. standard Markdown lexer cannot make Code Tree ok, it lists all headers in the same tree level, while this plugin makes nested nodes in the tree. it considers Markdown code-blocks, which have lines like headers.
